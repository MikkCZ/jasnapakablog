Jak z Firefoxu odstranit Microsoft .NET Framework Assistant

Pomocí automatických aktualizací Windows nabídl Microsoft na konci ledna aktualizaci .NET Frameworku 3.5 na verzi SP1. V rámci ní se uživatelům Firefoxu automaticky nainstalovalo rozšíření Microsoft .NET Framework Assistant 1.0. Jedná se o rozšíření, které do Firefoxu přidává podporu pro distribuci XBAP aplikací Windows Presentation Foundation (WPF). Pokud pomineme, že se rozšíření nainstaluje bez vědomí uživatele, je tu ještě jeden háček – rozšíření totiž nejde standardně odinstalovat. V seznamu nainstalovaných rozšíření ve Správci doplňků je tlačítko pro odinstalaci u uvedeného rozšíření jednoduše potlačeno.

Na blogu Firefox Extension Guru se však objevil návod, jak se uvedeného rozšíření zbavit. Jednoduše ukončete Firefox, přejděte do složky C:\Windows\Microsoft.NET\Framework\v3.5\Windows Presentation Foundation\DotNetAssistantExtension a obsah uvedené složky vymažte. Poté spusťte Firefox a rozšíření bude pryč. Nainstalované rozšíření taktéž drobně modifikuje user-agent řetězec prohlížeče, ve kterém naleznete text typu "(.NET CLR 3.5.30729)". Nejedná se o nic kritického, ale pokud chcete odstranit i to, přejděte na konfigurační stránku about:config a nastavte předvolbu general.useragent.extra.microsoftdotnet na výchozí hodnotu (nad předvolbou v místní nabídce volbou Obnovit).

Přečtěte si také:

  • MozBackup 1.4.9
  • Vyšel Thunderbird 3 Beta 2 i s novým webem
