Safari 3 Beta pro Windows

Dlouhým spekulacím o tom, zda Apple přijde se svým webovým prohlížečem Safari ve verzi pro Windows, bylo dnes učiněno přítrž. Steve Jobs (CEO Apple) dnes na Apple Worldwide Developers Conference v San Francisku oznámil dostupnost Safari 3 Beta pro Windows. Pro Windows je tak k dispozici další webový prohlížeč. Nedalo mi to, abych si jej nevyzkoušel. Na domovské stránce je k dispozici ke stažení instalátor o velikosti 8,4 MB. Dostupná je i verze s přehrávačem Quicktime. Safari je k dispozici "pouze" pro Windows XP a Vistu, takže uživatelé starších systému jako Windows 2000 mají smůlu.

Safari pod Windows XP působí graficky poněkud nepřirozeně. Vzhled stránek je vyhlazovaný a se zobrazováním některých textů jsou problémy, takže se vám může stát, že namísto textu uvidíte dlouhé podtržítko. Dost možná tím, že mam anglickou verzi systému s českým nastavením. Prohlížeč působí hodně nestabilním dojmem. Jakýkoliv pokus o přístup do historie či záložek u mě způsobí pád prohlížeče. Počítám, že uvedené nedostatky jsou dány tím, že se blížila konference a vývojáři museli představit nějakou, byť hodně nedodělanou, verzi. Škoda, řadu uživatelů tyto prvotní chyby asi odradí.

Když jsem si tak procházel možnosti prohlížeče, pak jsem marně hledal něco, čím by mě Safari zaujal. Prostě nějaká šikovná funkce, kterou jinde nenajdu a kvůli které je Safari uživateli Mac OS X oblíben. Bohužel, na nic jsem nenarazil. Nemohu se zbavit dojmu, že Safari je pod Mac OS X populární díky své dobré integraci se systémem. Uživatelé Mac OS X nechť prominou uživatele neznalého tohoto systému.

Safari je tu pro Windows a toť asi zpráva dne. Pevně doufám, že vývojáři do finální verze odstraní hlavní nedostatky a vytvoří tak další kvalitní prohlížeč pro Windows. Zda si své uživatele najde, je otázkou, protože nenabízí nic unikátního. Vývojáře však potěší, že mohou své webové stránky testovat na Safari pod Windows – byť samozřejmě nenasimulují stejný vzhled jako pod Mac OS X. Zájemci stahujte a zkoušejte.
