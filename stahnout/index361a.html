Doplňky / Rozšíření / Motivy vzhledu (skiny)

Prakticky všechny aplikace Mozilla si můžete rozšířit o nové funkce jednoduchou doinstalací doplňků. Jejich nejrozsáhlejší databázi poskytuje server Mozilla Add-ons.

Zálohování záložek, pošty, kontaktů...

Potřebujete snadno zazálohovat vaše záložky, poštu či kontakty z Firefoxu či Thunderbirdu? Použijte MozBackup, pomocí kterého snadno zazálohujete a obnovíte vaše data.

Zásuvné moduly (pluginy)

Řada stránek ke korektnímu zobrazení vyžaduje zásuvné moduly, které musí mít uživatel nainstalován. Nejčastěji se jedná o Flash, Javu či zásuvný modul přehrávače Windows Media Player. Pakliže vám nějaký z nich chybí, stáhněte si jej a nainstalujte klepnutím na odkaz níže.

  • Flash Player
  • Shockwave Player
  • Java

Řadu chybějících zásuvných modulů nabídne Firefoxu sám ke stažení a instalaci, případně odkáže na místo, odkud jej lze stáhnout a nainstalovat. Pokud vám nejde na stránce přehrávat video, doporučuji postupovat dle návodu.

Kontrola pravopisu

Firefox, Thunderbird a další aplikace Mozilla obsahují funkci kontroly pravopisu. Pro její korektní funkčnost je potřeba mít nainstalovaný slovník pro kontrolu pravopisu, který není z licenčních důvodů přímou součástí instalace. Můžete si jej ale nainstalovat přímo ze serveru Mozilla Add-ons.
