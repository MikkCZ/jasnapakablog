Windows Media Player plugin pro Firefox

Společnost Microsoft uvolnila Windows Media Player plugin pro Firefox. Jedná se primárně o řešení pro uživatele Firefoxu, kteří jej provozují pod operačním systémem Windows Vista. V něm totiž instalace Windows Media Playeru neobsahuje plugin pro spolupráci s prohlížeči jako Firefox či Opera, což se projevuje nemožností přehrát video na webových stránkách či poslechnutí rádia přes web. Instalaci uvedeného pluginu jsem zkoušel a příjemně mě překvapilo, že nalezla mou instalaci Firefoxu, i když není ve výchozím umístění tj. v cestě C:\Program Files\Mozilla Firefox. Uvedený plugin (zásuvný modul) je určen kromě Windows Vista i pro Windows XP SP2. Pokud tedy máte problém s přehráváním videa z důvodů nenalezení odpovídajícího pluginu, doporučuji vyzkoušet jeho instalaci. Po instalaci je samozřejmě dobré Firefox ukončit a opět spustit. Plugin je ke stažení ze stránek Technet.com. Pokud používáte prohlížeč z balíku SeaMonkey či vám uvedený plugin z nějakého důvodu nefunguje, doporučuji následující řešení problémů či manuální řešení pro Windows Vista.
