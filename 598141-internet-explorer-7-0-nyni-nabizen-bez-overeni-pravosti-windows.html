Internet Explorer 7.0 nyní nabízen bez ověření pravosti Windows

Pokud jako operační systém používáte Windows XP či Vistu, pravděpodobně jste se setkali s ověřením pravosti instalace tzv. Windows Genuine Advantage. V rámci tohoto programu se Microsoft snaží znepříjemnit život uživatelům nelegálních instalací Windows a naopak zvýhodnit ty, kteří mají řádně zakoupenou licenci. Bez ověření pravosti si tak nemůžete stáhnout některé ne-bezpečnostní aktualizace systému či doplňky. S ověřením pravosti systému jste se mohli setkat i v rámci instalátoru Internet Exploreru 7.0.

Dnes se na IEBlogu objevila zajímavá informace, že instalace Internet Exploreru 7.0 pro Windows XP již nebude vyžadovat potvrzení legálnosti systému. Jinak řečeno, Internet Explorer 7.0 nyní půjde nainstalovat i do nelegálně nainstalovaného systému. Poměrně zajímavý krok ze strany Microsoftu, který se na jednu stranu snaží zvýhodnit legální instalace systému, na druhou stranu říká "teď dáváme možnost instalovat IE 7.0 i uživatelům nelegálních instalací". Oficiálním zdůvodněním tohoto kroku je vyšší ochrana ekosystému Windows, což je poměrně pochopitelné zdůvodnění, protože Internet Explorer 7.0 je oproti verzi 6.0 bezpečnostně někde úplně jinde a v tomto směru udělali v Microsoftu velký kus práce. Nabízí se však otázka, proč k tomuto kroku došlo až nyní a zda není spíše důsledkem toho, že Internet Explorer 7.0 nenahrazuje předchozí verzi tak rychle, jak by si v Redmondu přáli.
