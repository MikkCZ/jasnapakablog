Firefox bude možná vyžadovat minimálně Windows XP SP3

Firefox 3 pro Windows ke svému chodu vyžaduje minimálně Windows 2000. Dřívější verze běžely oficiálně i na Windows 98/ME, ale právě počínaje Firefoxem 3 došlo k navýšení minimální konfigurace. I když existují způsoby, jak rozchodit Firefox 3 i na Windows 98, není to rozhodně doporučený způsob. Pro verzi, která vyjde po Firefoxu 3.5, nyní padl návrh na navýšení minimální verze Windows na Windows XP SP3. Mezi důvody tohoto kroku je uváděno zjednodušení testování, absence nutnosti potýkat se s chybami starších verzí Windows a hlavně fakt, že 13. července 2010 společnost Microsoft ukončuje podporu Windows 2000. Prozatím je návrh přijímán spíše rozporuplně, takže není jasné, zda k němu skutečně dojde.
