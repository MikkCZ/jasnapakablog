Firefox 3.0 a ukládání záloh záložek do formátu JSON

Jednou z posledních novinek, kterou zbývalo dopracovat pro připravovaný Firefox 3.0, je ukládání záloh záložek do formátu JSON (JavaScript Object Notation). Jedná se o jazykově nezávislý datový formát, který počínaje Firefoxem 3.0 nahradí ukládání záloh záložek. Ty se ve Firefoxu 2.0 ukládají v uživatelském profilu každý den do podadresáře bookmarkbackups a dosud byly ve formátu HTML. Nově tedy budou ukládány do formátu JSON, který má dle slov tvůrců zajistit lepší informační bezztrátovost záloh.

Možnost zazálohovat do formátu HTML zůstane zachována a to s ohledem na možnost importu vyexportovaných záložek do jiných aplikací, které původní HTML formát se záložkami umí importovat. Pokud se ve vývojové verzi Firefoxu 3.0 podíváte do Library (dříve Správce záložek), uvidíte v nabídce Import and Backup volby Export (slouží k exportu do HTML formátu) a Backup (export do formátu JSON). Spolu s nabídkami Import a Restore mi to ale přijde trochu nepřehledné, protože mi příjde, že se tyto volby funkčností v zásadě překrývají.

Přečtěte si také:

  • MozBackup 1.4.8 Alfa
  • Ohlasy na uvolnění Mozilla Prism 0.9
